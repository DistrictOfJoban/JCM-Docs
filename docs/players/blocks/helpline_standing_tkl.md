A **Helpline (Tseung Kwan O Line Type)** is a block in Joban Client Mod acting as a machine to request assistance from station staff. 

This replicates the Help Line seen in stations along the MTR Tsueng Kwan O Line.

## Usage
Right clicking the block would emit a redstone signal for 20 in-game ticks (or 1 second).
